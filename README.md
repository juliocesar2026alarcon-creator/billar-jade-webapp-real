
# 🎱 Billar Jade – Sistema Web de Control de Billar

Aplicación web profesional para la gestión completa de un billar real.

## Funciones obligatorias
- Control de mesas en tiempo real
- Cobro automático por tiempo con fracciones y redondeo
- Pausar, cerrar y pagar mesas
- Consumos por mesa
- Mover consumos entre mesas
- Descuentos sólo con PIN de administrador
- Impresión de tickets (PDF / impresora térmica)
- Caja diaria (apertura, movimientos, cierre)
- Inventario con Kardex
- Reportes por fecha, mesa, producto y cajero
- Usuarios con roles
- Multi‑sucursal
- Torneos de billar

## Tecnologías sugeridas
- Backend: Node.js + Express + PostgreSQL
- Frontend: React + Vite
- Hosting: Render
